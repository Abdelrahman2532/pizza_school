# Pizza School - Professional Restaurant Website Redesign

## Summary

The Pizza School website has been completely redesigned to be a **REALISTIC, PROFESSIONALLY-DESIGNED restaurant website** inspired by pizzahut.me and modern food delivery apps.

**NOT an AI-generated template.** This is a handcrafted, professional design that focuses on user experience and food ordering.

## Design Principles Applied

### 1. Professional Color Palette
```
Primary Black:    #111111 ← Serious, food-focused
Pizza Red:        #d62828 ← CTAs, brand recognition  
White:            #ffffff ← Clean backgrounds
Gray Scale:       #f3f4f6 to #9ca3af ← Subtle variations
```

### 2. Subtle Interactions
- ✅ Smooth transitions (200ms)
- ✅ Shadow increase on hover
- ✅ No scale transforms (too bouncy)
- ✅ Clean modal open/close
- ✅ Professional button states

### 3. Clean Typography
- **Font**: Inter (Google Fonts) - professional, readable
- **Sizes**: Responsive 4xl → 5xl for headings
- **Weights**: 400, 600, 700, 800
- **Spacing**: Proper line height (1.5x for body)

### 4. Focus on Ordering
- Category filtering (9 categories)
- Full-text search
- Size selection with real pricing
- Quantity control
- Promo code support
- Clear order summary
- Professional checkout flow

## Website Structure

### 1. Navigation
```
[LOGO] [Menu] [Contact] [Search] [Cart Counter]
```
- Sticky position with minimal shadow
- Clean, professional appearance
- Mobile: Hamburger menu
- Cart icon shows item count

### 2. Hero Section
```
[TEXT & BUTTONS] | [IMAGE PLACEHOLDER]
```
- Strong typography: "Premium Pizza"
- Value proposition
- Two CTAs: "Order Now", "View Menu"
- Info grid: 30m delivery, Fresh ingredients, 10+ years
- Two-column responsive layout

### 3. Menu Section
```
[Category Tabs] → Filter by type
[Search Bar] → Find items
[Product Grid] → 4-column responsive
```

**Menu Features**:
- 70+ items across 9 categories
- Search by name or description
- Best Seller / New badges
- Spicy level indicators
- Starting prices from menu data

**Category Tabs**:
- Horizontal scrollable on mobile
- Sticky red highlight for selected
- Smooth category switching
- No animations, just clean switches

### 4. Product Cards
```
┌─────────────────┐
│                 │
│   Image         │ ← Gray placeholder
│   Placeholder   │   (ready for real images)
│                 │
├─────────────────┤
│ Name            │
│ Description     │
│ 🌶️ Spicy Level │
│ From $12.99     │
└─────────────────┘
```

### 5. Product Modal
```
[Close X]

[IMAGE] [DETAILS]
  Title
  Description
  Ingredients (chips)
  Size Selector (M, L, F, XXF)
  Quantity (+/-)
  Price Calculation
  Add to Cart Button
```

### 6. Shopping Cart

**Mobile**: Drawer from bottom-right
- Slides out from right side
- Full width on small screens
- Item list
- Promo code input
- Order summary
- Checkout button

**Desktop**: Fixed sidebar
- Always visible on right
- 320px wide
- Same content as mobile
- No drawer animation

- Dark background (#111111)
- Professional, minimal
